Expand description

Encodes and attempts to write an u64 value strictly as a 9-byte sequence into the given write.

The first byte becomes the marker and the others will represent the data itself.

Note, that this function will encode the given value in 9-byte sequence no matter what, even if the value can be represented using single byte as a positive fixnum.

If you need to fit the given buffer efficiently use write_uint instead, which automatically selects the appropriate integer representation.

This function will return ValueWriteError on any I/O error occurred while writing either the marker or the data.
